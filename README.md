# Activity Tracker

Local only, manual entry activity logging program

# Development
Set up virtual environment:

## Linux + macOS
```
python3 -m venv venv
source venv/bin/activate
python3 -m pip install -r requirements.txt
```

## Windows
```
python3 -m venv venv
venv\Scripts\activate
python3 -m pip install -r requirements.txt
```
